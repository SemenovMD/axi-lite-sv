# AXI Lite to AXI Stream Converter

Данный модуль представляет собой конвертер между интерфейсами `AXI Lite` и `AXI Stream`. Он позволяет преобразовывать транзакции AXI Lite в потоковые данные AXI Stream и обратно, обеспечивая взаимодействие между процессорными системами и потоковыми обработчиками данных.

## Особенности

- Операции записи и чтения через AXI-Lite.
- Настраиваемая ширина данных и адреса.

## Параметры
- `AXI_DATA_WIDTH` — ширина данных AXI-Lite (по умолчанию 32).
- `AXI_ADDR_WIDTH` — ширина адреса AXI-Lite (по умолчанию 32).

## Документация

### `axil_converter_str`
Основной модуль конвертера `AXI-Lite` в `AXI-Stream`.

### `axil_if`
Интерфейс AXI-Lite для подключения к внешней шине.

### `axis_if`
Интерфейс AXI-Stream для подключения к внешней шине.

### `axil_converter_str_wrapper_sv`
Промежуточная оболочка на SystemVerilog для последующей обертки в Verilog.

### `axil_converter_str_wrapper_v`
Оболочка на Verilog для удобного использования в блок дизайне (IP Integrator).
